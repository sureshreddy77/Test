Git Command:-

1   git init                                     -------> Initializes empty Git repository
2   git config --global user.name "sureshreddy77"     -------> Git initialization
3   git config --global user.email "sreddy5482@gmail.com" -------> Git initialization
    git clone https://github.com/sureshreddy77/Test.git
4   git add filename                             -------> It will adds the file untracked to tracked stage.
5   git add .                                    -------> Same as git add but if you execute this command all the files will added here in a single shot.
6   git status                                   -------> It will displays the status of the files
7   git commit filename                          -------> To track the files
8   git commit -m "Initial commit"               -------> Same as git commit but in single shot we can commit all the files
9   git remote add origin "repo URL "            -------> To map our local repo to remote repo
10  git push -u origin master                    -------> It will push the code the remote repo
11  git branch branch-name                       -------> To create the branch name
12  git branch –v                                -------> It will display branches.
13  git checkout branchname                      -------> It will point the new branch
14  git log                                      -------> It will shows all the commit id and other info
15  git revert commitid                          -------> To revert back
16  git stash                                    -------> It will saves your working job and makes the job working area clean
17  git stash pop                                -------> It will revert backs to the job 
18  git stash apply                              -------> It will also reverts back to the job but it will keep an  copy in stash
19  git stash list                               -------> It will list the files in the stash
20  git stash drop stashed                       -------> To the file from the stash
21  git stash clear                              -------> It will delete all the files from the stash
22  git remote –v                                -------> It will display the repo URL
23  git pull repo                                -------> It will downloads the code to local repo(git fetch+git merge)
24  git fetch                                    -------> It will push code from remote to local repo
25  git commit -a -m "Commit name"               -------> If we use this command then no need to add command 
26  git clean -n                                 -------> It will display the preview changes
27  git clean -f                                 -------> It will deletes the files from the working area
28  git reset                                    -------> To untrack the tracked files
29  git diff branch1 branch2                     -------> It will show the difference between those two branches
30  git merge master branchname                  -------> It will merge with the master branch
31  git branch -m oldname newname                -------> To rename the branch
32  git branch -a                                -------> It will display all remote and local branches
33  git branch -r                                -------> It will display all remote branches
34  git rm                                       -------> It will remove files from your index and working directory 
35  git checkout -b branchname                   -------> It will creates the branch and points to that branch
36  git push --delete branchname                 -------> It will delete the branch from the remote repo
37  git clone github URL                         -------> To clone the data from the remote repo to local
38  git push origin --all                        -------> It will pushes all branches code to your remote repo
39  git tag tagname                              -------> To create the tag
40  git push origin --tags                       -------> It will push the tag to remote repo
41  git tag                                      -------> It will displays the tags i.e release

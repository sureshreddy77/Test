1)What is Tomcat ?  
  - Tomcat is a java servlet container and web server from Apache Software Foundation
  - It can be used as standalone or it can be used as traditional web-server such as Apache httpd
  - It provides servelets and JSP Technologies
  - You can install 2 ways in Windows(.exc and extract zip file)
  - bin, lib, tmp, work, conf, logs and webapps
  - server.XML under conf
  - Admin Port:8005
  - AJP POrt: 8009
  - 2 connectors HTTP Connector and AJP Connector
  
2)Services Provided By Tomcat which are not provided by Apache httpd
  - Servelet Life Cycle
  - High Availability
  - Clustering
  - Handle Web Requests
  - Database connection Pooling
